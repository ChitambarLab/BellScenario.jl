Games represent a linear inequality.

BellScenario.Game — Type
Game(game::Matrix{T}, β::Real) <: AbstractMatrix{T}

A Game is played by a Matrix and β is the bounding or winning score of the game. A Game is matrix representation of a linear inequality. Each element is a linear scale factor for an element of a strategy matrix.

Type parameter T is typically an Int64 or Float64.
